# USC Data Science Coursework

A collection of projects and assignments completed during my graduate studies at the University of Southern California (USC).

## Courses Included

### DSCI 531: Data Visualization
Exploration of visual analytics using tools like D3.js, Tableau, and R/ggplot2.

### DSCI 553: Foundations and Applications of Data Mining
Implementation of algorithms for frequent itemsets (Apriori, SON), recommendation systems, clustering (BFR), and graph analytics.

### DSCI 560: Data Science Professional Practicum
Application of data science techniques to real-world problems, including IoT sensor data analysis and event detection.
